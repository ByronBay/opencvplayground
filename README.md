# opencvplayground
Accumulation of snippets and projects around opencv

## First run

* git clone https://github.com/ByronBay/opencvplayground.git
* python -m venv venv
* source ./venv/bin/activate
* pip install -r requirements.txt

## Further runs

* cd PROJECT_DIR
* source ./venv/bin/activate
* pip install -r requirements.txt
